PredictIt Arbitrage Engine

Submitted by: Srichinmai Sripathi

Project Summary

This engine automatically detects and calculates risk-free arbitrage opportunities on PredictIt.org. By leveraging linear programming, it identifies markets where buying "No" contracts on all outcomes results in a guaranteed profit, regardless of the election result.

How to Run

Clone the repository and run the following commands:

# Install dependencies
pip install -r requirements.txt

# Run the engine
python src/main.py

Output

The engine generates two CSV files:

  • predictit_markets.csv: A snapshot of all active markets and prices.
  • arbitrage_opportunities.csv: Detailed list of profitable trades found.
